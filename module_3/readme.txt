Основные цели и задачи проекта:

Цель - предсказать Рейтинг ресторанов TripAdvisor основываясь на предоставленных данных.


Задачи:

1. Исследовать предоставленный датафрейм
2. Найти пропуски/выбросы
3. Заполнить пропуски, убрать выбросы
4. На основе предоставленных данных ввести новые признаки, полезные для модели
5. Ввести внешние данные и на основе них ввести новые признаки и улучшить модель
6. Участвовать в соревновании Kaggle
